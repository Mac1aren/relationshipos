# RelationshipOS v0.2 β ♡ — мобильный сайт Милый boot-screen сайт под телефон с анимацией загрузки и кнопкой на Telegram (`@raam_est`). ## Что внутри папки - `index.html` — сам сайт (всё в одном файле: вёрстка, стили, анимация). - `wallpaper.jpg` — **сюда положи свои розовые обои** (см. ниже). Без файла сайт всё равно красивый — под ним лежит розовый CSS-фон. ## 1. Добавить свои обои 1. Сохрани ту самую розовую картинку-коллаж как `wallpaper.jpg`. 2. Положи её в эту же папку, рядом с `index.html`. 3. Готово — сайт сам подхватит `./wallpaper.jpg` как фон (поверх неё лёгкая розовая вуаль, чтобы текст читался). > Хочешь другое имя/формат (например `.png`) — поменяй строку `url('./wallpaper.jpg')` в `index.html` (блок `.wallpaper`). ## 2. Что можно быстро поменять - **Ссылка на Telegram** — в `index.html` найди `https://t.me/raam_est` и замени при необходимости. - **Текст лога** — это обычные строки `
…
` внутри `
`. Добавляй/удаляй строки как хочешь, анимация порядка подстроится сама. - **Цвета** — вверху в `:root` (`--pink`, `--ink`, `--mint` и т.д.). ## 3. Залить на GitHub Pages 1. Зайди на https://github.com → **New repository**. Имя любое, например `relationshipos`. Сделай его **Public**. 2. На странице репозитория → **Add file → Upload files** → перетащи `index.html` и `wallpaper.jpg` → **Commit changes**. 3. **Settings → Pages**. В разделе *Build and deployment*: Source = **Deploy from a branch**, Branch = **main**, папка **/ (root)** → **Save**. 4. Через 1–2 минуты сайт будет доступен по адресу `https://<твой-логин>.github.io/relationshipos/`. ## 4. Привязать свой домен У тебя уже есть купленный домен — привязка делается в два шага. **A. В GitHub:** 1. **Settings → Pages → Custom domain** → впиши свой домен (например `raam.example` или `os.raam.example`) → **Save**. GitHub сам создаст в репозитории файл `CNAME` с этим именем. 2. Позже включи галочку **Enforce HTTPS** (станет доступна после проверки DNS). **B. В панели управления доменом (там, где покупал) — добавь DNS-записи:** Если используешь **корневой домен** (`raam.example`) — добавь 4 A-записи: ``` A @ 185.199.108.153 A @ 185.199.109.153 A @ 185.199.110.153 A @ 185.199.111.153 ``` И желательно AAAA (IPv6): ``` AAAA @ 2606:50c0:8000::153 AAAA @ 2606:50c0:8001::153 AAAA @ 2606:50c0:8002::153 AAAA @ 2606:50c0:8003::153 ``` Если используешь **поддомен** (`os.raam.example` / `www.raam.example`) — одна запись: ``` CNAME os <твой-логин>.github.io ``` DNS обновляется от нескольких минут до пары часов. После — открой `Settings → Pages`, дождись «✓ DNS check successful», включи **Enforce HTTPS**. > Важно: имя в файле `CNAME` (внутри репозитория) и Custom domain в настройках должны совпадать с доменом, на который ведут DNS-записи. ## Проверка - Сайт открывается на телефоне на весь экран, текст разбит по строкам, появляется с анимацией, внизу — розовая кнопка «Open Telegram», ведущая на `https://t.me/raam_est`.
